Building my project, EXELocker, taught me the value of test-driven development. EXELocker.xyz

EXELocker is a free and open-source application that allows users to password-protect any executable (.exe) file using strong AES-256 encryption. This ensures that the locked files cannot be opened without the correct password, providing an extra layer of security.

  • Lock any executable program: Users can select any .exe file they want to lock and protect with a password.
  • Strong encryption: The application uses state-of-the-art AES-256 encryption to secure the locked files, making it virtually impossible for anyone to access them without the password.
  • Backup option: EXELocker offers a "Create Backup" option, which creates a copy of the original .exe file with a .old extension. This allows users to test the software and restore the original file if needed by simply renaming the .exe.old file to .exe.
  • User-friendly interface: The application has a simple and intuitive interface for both locking and unlocking files, making it easy for users to secure their executables.
  • Free and open-source: EXELocker is completely free to use without any hidden costs or adware. Additionally, the entire source code is available on GitHub, ensuring transparency and allowing users to verify the software's integrity.
